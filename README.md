# Codespace Linux GUI

## Quick Setup

Open the Codespace terminal and run:

```bash
bash /workspaces/codespace-linux-gui/setup.sh
```

This installs XFCE desktop + TigerVNC + noVNC natively.

## Access

- **GUI Desktop**: Ports tab → port 6080 → Open in Browser
- **Terminal**: `gh codespace ssh` or use the built-in terminal
